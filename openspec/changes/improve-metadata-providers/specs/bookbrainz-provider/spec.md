> **REMOVED.** BookBrainz was evaluated during implementation and removed entirely. The documented API endpoints (`/search/work`, `/ws/2`) do not match the actual REST API, the data is alpha-quality, and retrieving series data requires multiple round-trips with unreliable results. All tasks for BookBrainz integration are cancelled. No `BookBrainzService` exists in the codebase; it does not appear in `MetadataModule`, `PROVIDERS_CONFIG`, or any provider chain.
